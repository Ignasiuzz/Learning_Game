# Unity Learning Game

A 2D Unity 6 project focused on exploration, interaction, and item collection while completing simple learning tasks.  Built with a modular architecture for reusable gameplay components.

## Technologies Used

- **Unity 6**
- **Version 6000.2.7f2**

## Controls

| Key | Action |
|-----|---------|
| **E** | Interact with objects/NPCs |
| **Tab** | Toggle pause menu |
| **WASD / Arrow Keys** | Move player |
